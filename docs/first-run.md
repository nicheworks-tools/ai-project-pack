# First Run

1. Replace the example content in the five core files with your own project state.
2. Read `../README.md` and `../AGENTS.md`.
3. Start with report-only.
4. Approve only one bounded safe-update.
5. Add one dated log under `../updates/`.

## A good first loop feels like
- current truth gets sharper
- next actions get smaller and clearer
- unresolved items stay visible
